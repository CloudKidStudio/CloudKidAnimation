CloudKid Animation
=================

Animation API for CreateJS or Pixi

##Documentation##

http://cloudkidstudio.github.io/CloudKidAnimation/

##Installation##

CloudKid Animation can be install using Bower.

```bash
bower install cloudkid-animation
```
